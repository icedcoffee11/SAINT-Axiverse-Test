# Project Status

This repository contains a speculative frontier-physics proposal: SAINT, an interplanetary atomic-clock network concept for testing string-axiverse signatures.

- Main document: `README.md`

## Verification notes

This is not yet a peer-reviewed physics paper, mission study, or funded technical design. Before formal submission or outreach to experts, the following need careful checking:

1. Every physics citation should be verified against the original paper.
2. Equations should be checked by a domain expert in ultralight dark matter, atomic clocks, and string phenomenology.
3. Sensitivity estimates should be recalculated with explicit units and reproducible notebooks.
4. Claims about excluded fractions of string compactifications should be reframed as hypotheses unless backed by a concrete ensemble calculation.
5. Mission architecture assumptions should be separated from near-term terrestrial clock-network work.
6. Cost ranges and technical-readiness statements should be labelled as rough-order estimates unless sourced.
